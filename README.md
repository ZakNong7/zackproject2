ZackProject Landing Page
Deskripsi Proyek
Landing page untuk agensi kreatif zackproject dengan konsep clean, elegan, dan premium. Halaman ini dirancang untuk menampilkan layanan kreatif seperti desain brand, pengembangan web, dan pemasaran digital. Menggunakan tema warna hitam-gold untuk kesan mewah dan profesional.

Fitur Utama
Hero Section: Dengan efek parallax dan animasi smooth fade-in.
Portfolio Slider: Galeri gambar yang berganti otomatis atau manual dengan tombol prev/next dan swipe touch.
Daftar Layanan: Grid card dengan icon modern dan efek hover.
Testimoni Carousel: Slider otomatis dengan indikator dots, tombol navigasi, dan pause on hover.
Form Kontak: Dengan validasi JavaScript untuk input nama, email, dan pesan.
Navigasi Burger: Menu responsif yang muncul di mobile dengan animasi smooth.
Footer Profesional: Dengan link cepat, informasi kontak, sosial media, dan copyright.
Responsif: Dioptimalkan untuk desktop, tablet, dan mobile.
Teknologi yang Digunakan
HTML5: Struktur halaman.
CSS3: Styling dengan animasi, grid, dan media queries.
JavaScript: Interaktivitas seperti slider, carousel, validasi form, dan burger menu.
Font: Poppins dari Google Fonts untuk tipografi tegas.
Icon: Emoji untuk icon sederhana (bisa diganti dengan SVG/FontAwesome).
